Énergie Solaire

L'énergie solaire est déjà utilisé dans beaucoup d’endroits. Cettes endroit sont déserts, les toundras et autres paysages dénudés. En plus, les toit avec beaucoup de lumiere du soleil dans toute l’Amerique peuvent cree de tas d’energie . La quantité de soleil qui brille sur la Terre est énorme!




Énergie Nucléaire

Les gen est peur que produit l'énergie avec les centres nucléaire. Mais si nous sommes prudent, le’engie nucleaire est très sûr. Nous pouvons enterrer les centres nucleaire sous les montagnes ou dans la Terre.




Voitures Électrique

Les voitures hybrides et électriques sont les machines du futur. Parce que les voitures électriques utilisent entièrement les moteurs électriques, ils n’émettent pas un seul spec de charbon. Bien que les voitures electrique utilises seulement l’électricité, comment cette électricité produit est encore un problème.




La Viande Synthétique

Les scientifiques peuvent créer les vianda dans un laboratoire. Avec cette technologie, nous pouvons produire beaucoup de viande bon marche. C’est important pour les pays qui ont les probleme avec nourriture mais l'avantage primaire est reduire l’emission de méthane de bétail.
